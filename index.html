About
Welcome to our Minecraft Server

This is a vanilla-style Minecraft server with a few quality-of-life plugins. We've been running since 2012 and have a friendly community of builders and explorers.

Server IP: mc.example.com

Version: 1.20.1

Rules: No griefing, no cheating, be respectful to other players.

Server Features
  • Survival mode with land claiming
  • Monthly building contests
  • Economy system with player shops
  • Regular world backups
  • Active staff team
FAQs
How do I join the server?
Simply connect to mc.example.com using Minecraft Java Edition version 1.20.1. No whitelist is required.
How do I claim land?
Use a golden shovel to select two corners of the area you want to claim, then type /claim create. You get 100 claim blocks when you first join, and earn more by playing.
Are there any banned items?
TNT and other griefing tools are restricted. See the full list with /rules in-game.
